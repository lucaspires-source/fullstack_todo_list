

# Todo Fullstack

Uma aplicação  fullstack com React, Node.js, TypeScript

## Iniciando

1. **Clone o repositório**
   ```bash
   git clone https://github.com/your-repo/todo-fullstack.git
   cd todo
   ```

2. **Execute o Front-end**
    ```bash
    cd todo-frontend
    npm run dev
    ```

3. **Execute o Back-end**
    ```bash
    cd todo-backend
    npm run dev
    ```
